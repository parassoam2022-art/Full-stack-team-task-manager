# Full Stack Team Task Manager

## Overview
A full stack web application for managing team tasks, tracking progress, and improving collaboration between team members.

## Features
- User authentication
- Create and manage tasks
- Assign tasks to team members
- Track task status and progress
- Responsive dashboard

## Tech Stack
- Frontend: React.js
- Backend: Node.js & Express.js
- Database: MongoDB

## Installation

```bash
git clone https://github.com/parassoam2022-art/Full-stack-team-task-manager.git
cd Full-stack-team-task-manager
npm install
npm start
